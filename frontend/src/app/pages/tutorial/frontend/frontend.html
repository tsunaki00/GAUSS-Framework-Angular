GAUSS Framework tutorial!!

はじめに

チュートリアルの目標
簡単なTodoアプリケーションを作成する。
画面イメージ
[機能一覧]
機能NO 機能名 機能概要
0001 Todo一覧表示 登録したTodoの一覧を表示します。
0002 Todo登録 入力情報でTodoを登録します。
0003 Todo削除 チェックしたTodoを削除します。(自習)
[前提条件]
・開発ポートは4200番を使用します。
・Backendのサーバー開発完了している前提とします。(以下API一覧）
※※ Backendのサーバが起動している状態で実行してください。
No API URI Parameter Method description
1 ToDo一覧取得 /rest/todo GET ToDoリストを取得する
2 ToDoの登録 /rest/todo ｛ "message" : "example" ｝ POST ToDoを登録する
3 ToDoの削除 /rest/todo/:id DELETE ToDoを削除する
[開発に必用なスキル]
  • HTML5
  • CSS3（sass）
  • Javascript（ECMAScript 6～）
