  • STEP 1　事前準備

  • ① 必用なミドルウェアのインストール

    [nginx]
    1. 以下のサイトからインストール用のパッケージをダウンロードして手順に従ってインストールしてください。
    [brew]
    https://brew.sh/index_ja
    # brewのインストール
    $ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
    2. terminalで以下のコマンドを実行してnginxをインストールします。
    # ngnxのインストール
    $ brew install nginx
    $ nginx -v nginx
    version: nginx/x.xx.x
    [docker]
    1. 以下のサイトからインストール用のパッケージをダウンロードして手順に従ってインストールしてください。
    https://docs.docker.com/docker-for-mac/install/
    [docker compose]
    2. 次にdocker-composeをインストールします。
    こちらも以下のサイトを参考にインストールしてください。
    https://docs.docker.com/compose/install/
    関連
    docker, docker(Wikipedia), docker-compose
    -- Memo --
    開発をする際は以下の開発エディタを使用することを勧めます。
    Visual Studio Code
  • STEP 2 実行環境の構築(Frontend)

  • ① リポジトリからソースの取得

    1. terminalで以下のコマンドを実行してプログラムを取得します。
    # 開発用のworkspaceに移動
    $ git clone https://github.com/tsunaki00/GAUSS-Framework-Angular.git
    $ cd GAUSS-Framework-Angular
    関連
    git, git(Wikipedia)

  • ② 開発用のdockerコンテナの起動

    1. terminalで以下のコマンドを実行してdockerを起動します。
    ※初回の起動は時間がかかります
    $ docker-compose up -d
          : 
    Starting gaussframeworknosql_mongodb_1 ... done
    Starting gaussframeworknosql_mongo-express_1 ... done
    2. dockerコマンドでUPしたことの確認を行えます。
    ※ 起動中のコンテナの一覧が表示されます
    $ docker ps
          : 
          : 
    ※※ 停止したい場合はdocker-composeがあるディレクトリで以下のコマンド実行します。
    $ docker-compose down
          : 
    Stopping mongodb ... done
    Stopping mongo-express ... done
    Removing mongodb ... done
    Removing mongo-express ... done

  • ③ フロントエンドの実行で必用なパッケージのインストール

    # forntendに移動

  • ① フロントエンドとバックエンドの連結にNginxにリバースプロキシを設定します。

    1. terminalで以下のコマンドを実行して設定ファイル更新します。 ([INSTALL_DIR]/nginx])
    # 設定ファイルを更新します
    $ cd ../nginx
    $ cp –p /usr/local/etc/nginx/nginx.conf /usr/local/etc/nginx/nginx.conf.org
    $ cp –p ./nginx.conf /usr/local/etc/nginx/nginx.conf

  • ② nginxの設定をリロードします。

    1. nginxの設定をリロードします。
    ※初回の起動は時間がかかります
    $　nginx –s reload
